Usage

1. Name the type
2. Paste JSON
3. Click the green Convert button
4. Your Go code will appear in the Golang tab

What's This All About?

This tool converts JSON into a rough Go type definition. Useful when you want to avoid spelling out each field manually or using interface{} and type assertions for JSON deserialization. The translater makes some assumptions, so be sure to check the results (and then use go fmt.)
