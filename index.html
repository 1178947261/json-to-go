JSON-to-Go

Convert JSON to Go struct

This tool converts JSON into a rough Go type definition. Paste the JSON structure and the equivalent Go type will be generated, which you can paste into your program. Don't forget to give it a once-over, since this script has to make some assumptions. (And go fmt your code!)

For an example, try converting JSON from the SmartyStreets API or the GitHub API.
